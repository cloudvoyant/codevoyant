# push-comments

Read a review doc (generated by `/rev new` or hand-written) and submit each entry as an inline comment on the PR. Comments are pushed as a single PENDING review — publish with `/gh complete`.

## Arguments

- `PR_NUMBER` (optional positional) — defaults to PR for current branch
- `--doc <path>` — review doc path (default: `.codevoyant/review/{branch}/new-review.md`)

Expected doc format: see `skills/rev/references/new-review-template.md`. Each comment has: file path, line, body, optional suggestion block.

## Step 1: Verify `gh` CLI

```bash
command -v gh >/dev/null 2>&1 || { echo "gh not installed. Run: brew install gh"; exit 1; }
gh auth status >/dev/null 2>&1 || { echo "gh not authenticated. Run: gh auth login"; exit 1; }
```

## Step 2: Resolve PR & Repo

Detect `PR_NUMBER` from branch if not given (see `pull-comments.md` Step 2). Then:

```bash
REPO=$(gh repo view --json nameWithOwner --jq '.nameWithOwner')
OWNER="${REPO%/*}"
NAME="${REPO#*/}"
```

## Step 3: Parse Review Doc

Read `--doc` path. Parse each comment entry. Each comment block yields:

- `file` — file path relative to repo root
- `line` — integer line number in the file (right side of diff)
- `body` — Markdown comment body (may include a fenced ` ```suggestion ` block)

Validate every entry has `file`, `line`, `body`. Abort with clear error if any are missing.

## Step 4: Submit as Pending Review

Batch all comments into one review object via `gh api`:

```bash
gh api "repos/${OWNER}/${NAME}/pulls/${PR_NUMBER}/reviews" \
  --method POST \
  --field event=PENDING \
  --field body="" \
  --field "comments[][path]={file}" \
  --field "comments[][line]={line}" \
  --field "comments[][body]={body}"
```

Build the `comments[]` repeated fields by iterating the parsed entries.

Capture the returned review id:

```bash
REVIEW_ID=$(gh api ... --jq '.id')
```

## Step 5: Report

```
✓ {count} comments pushed as pending review (id: {review-id}). Run /gh draft or /gh complete to submit.
```

## Error Handling

- **Missing doc:** report path and exit
- **Malformed entry:** print the offending block and exit before any API call
