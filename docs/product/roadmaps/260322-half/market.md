### Existing Solutions and Gaps
- **IDE agents (Cursor, Windsurf, Copilot)**: Strong at inline editing, weak at orchestration and autonomous workflows
- **Terminal agents (Claude Code, Codex CLI, Aider)**: Good for hard problems, lack multi-agent coordination and monitoring
- **Orchestrators (agent-orchestrator, CAO)**: Emerging category, fragmented, no dominant player
- **Cloud platforms (Vercel, Railway, Fly.io)**: Framework-specific or require DevOps knowledge, none integrate with coding agent workflows
- Gap: No unified tool combines coding agent capabilities with deployment platform awareness
- Gap: Open-source agent ecosystem lacks standardized skill/plugin interoperability (Codevoyant's portable-agent-skills plan addresses this)
